# FitHub – Online Fitness Website 💪

FitHub is a full-stack online fitness website developed using Python, Django, and SQLite. The platform provides separate authentication and dashboards for users, trainers, and admin with OTP-based login functionality. Users can explore workout categories, track fitness progress, purchase memberships, and interact with the fitness community, while trainers and admin can manage workouts, schedules, clients, memberships, and overall platform activities efficiently.

## 🚀 Features

### 🔐 Authentication System
- OTP-based Login & Registration
- Separate User, Trainer & Admin Authentication
- Secure User Management

### 🏋️ Workout Categories
- Yoga
- Strength Training
- Cardio
- HIIT Workouts
- Weight Loss Programs
- Diet Plans

### 👨‍🏫 Trainer Dashboard
- Trainer Profile Management
- Add / Update / Delete Workouts
- Schedule Workout Sessions
- View Assigned Clients
- Track Trainer Earnings
- Manage Workout Plans

### 👤 User Features
- Membership Plans
- Fitness Progress Tracking
- Community Page
- Contact Page
- Workout Session Details
- Trainer Information Display

### 🛠️ Admin Dashboard
- Manage Users & Trainers
- Approve or Remove Trainers
- Manage Membership Plans
- Manage Workout Categories
- Monitor Platform Activities
- View Reports & Analytics

## 🛠️ Technologies Used
